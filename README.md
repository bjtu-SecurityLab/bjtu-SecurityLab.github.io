# Welcome to bjtu-SecurityLab

We are a security research team focused on **software security** and **vulnerability discovery**.

## Our Projects

- **AgentHive**: A core system for vulnerability analysis.
- **FirmHive**: A firmware security analysis application of AgentHive.

## How to Use

To use our tools, follow these steps:
1. Clone the repository.
2. Install dependencies.
3. Run the analysis.
